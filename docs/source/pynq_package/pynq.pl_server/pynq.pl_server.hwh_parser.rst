pynq.pl_server.hwh_parser Module
================================

The pynq.pl_server.hwh_parser module parses the metadata file (hwh file).
The goal is to extract useful information about the bitstream, 
including IP parameters, system clocks, block hierarchy, interruputs, 
and many others. This is the recommended way of getting overlay information.
